\# Poultry Farm Management System



A poultry farm system composed of two connected applications:



\# 1. Egg Trading and Grading System

\- PHP

\- MySQL

\- Bootstrap

\- Runs on XAMPP



\## 2. Inventory API System

\- Laravel

\- MySQL

\- REST API



\# System Flow



Egg Trading System sends graded egg records to the Laravel Inventory API using JSON-based HTTP requests.



\# Main Features



\- Admin login

\- Egg batch management

\- Egg grading

\- Egg sales

\- Customer management

\- Inventory API integration

\- Reports
